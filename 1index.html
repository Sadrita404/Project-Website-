How to Make a Robotic Arm

By Sadrita Neogi

This project uses the NodeMCU 1.0 ESP8266. I chose this board because it has built-in WiFi, meaning you don't need external Bluetooth modules to connect to your phone. It's the most efficient choice for wireless robotics.

Project Overview Image

Step 1: Components

  • NodeMCU 1.0 ESP8266 Board (x1)
  • Servo Motors (x4)
  • Dotted Veroboard / Custom PCB
  • 3D Printed Arm / Acrylic Sheets
  • Nuts, Bolts, and Headers
Component List Image

Step 2: Wiring & Connections

Servo VCC -> NodeMCU Vin (5V)
Servo GND -> NodeMCU GND
Servo 1 -> Pin D5
Servo 2 -> Pin D6
Servo 3 -> Pin D7
Servo 4 -> Pin D8
Wiring Diagram

Step 3: PCB & JLCPCB

I designed the PCB and ordered it from JLCPCB. Their quality is industrial-grade and very affordable—10 PCBs for just $2!

PCB Design View

Step 4: RemoteXY Interface

Using RemoteXY, I built a mobile interface to control the arm without writing complex app code. Install the library in Arduino IDE and upload your code!
