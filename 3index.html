Cheap Arduino Controlled 3-Axis Pen Plotter

By Sadrita Neogi

Introduction

The purpose of this project was to make a 3-axis pen plotter as cheaply as I possibly could. Assuming you already have an Arduino, it ended up costing around $30 to build.

Note: Stepper motors inside CD-ROMs are not very strong. If the pen comes down too hard, it will stall the motors. It won't hurt them, but it’s a good thing to keep in mind!

1
Supplies & Parts List

If you don't already have an Arduino, the total cost will be closer to $60.00.

2
Mechanical Construction

The build utilizes the stepper motors and sliding rails from three old CD-ROM drives. Each drive handles one axis (X, Y, and Z).

3
Wiring & Schematic

The schematic followed for wiring the SN754410NE H-Bridge drivers is crucial. I used an LM317 5V regulator for each bridge, though if you use a stable PC power supply, you can wire 5V directly.

4
The Arduino Code

The program reads binary 0's & 1's stored in a word array. You can draw different shapes by editing the image array. 

// Cheap 3-Axis Pen Plotter Code
#include <Stepper.h>

#define STEPS 200

Stepper stepperX(STEPS, 6, 7, 8, 9);     // x platform
Stepper stepperY(STEPS, 2, 3, 4, 5);     // y platform
Stepper stepperZ(STEPS, 10, 11, 12, 13); // z pen (Up/Down)

int x_inc = 0;
int y_inc = 0;
int z_pos = 0;

word image[16] = {
  0b0000001110000000,
  0b0010011111000100,
  0b0010001110000100,
  0b0010000100000100,
  0b0011111111111100,
  0b0000011111000000,
  0b0000011111000000,
  0b0000001110000000,
  0b0000011111000000,
  0b0000011111000000,
  0b0000110001100000,
  0b0001100000110000,
  0b0001100000110000,
  0b0011100000111000,
  0b0011100000111000,
  0b1111111111111111
};

void setup() {
  stepperX.setSpeed(60); 
  stepperY.setSpeed(60); 
  stepperZ.setSpeed(60); 
  delay(2000);
}

void loop() {
    for(int row = 0; row < 16; row++) {
      penUp();
      stepperX.step(-x_inc);
      x_inc = 0; 

      for(int column = 0; column < 16; column++){
        boolean pixel = bitRead(image[row],column);
        if(pixel == 1){
          penDown();
          stepperX.step(10);
          x_inc = x_inc + 10;
          delay(15);
        } else {
          penUp();
          stepperX.step(10);
          x_inc = x_inc + 10;
          delay(15);
        }
      }
      penUp();
      stepperY.step(10);
      y_inc = y_inc + 10;
      delay(15);
    }
  penUp();
  stepperX.step(-x_inc);
  stepperY.step(-y_inc);
  while(1){}
}

void penDown() {
  if (z_pos == 0){
    stepperZ.step(83); 
    delay(15);
    z_pos = 83;
  }
}

void penUp() {
  if(z_pos == 83){
    stepperZ.step(-83);
    delay(15);
    z_pos = 0;
  }
}
